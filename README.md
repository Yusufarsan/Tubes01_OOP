# Tubes01_OOP

Tugas Besar OOP

## Cara Mengkompilasi

1. Install CMake

    ```bash
    sudo apt-get -y install cmake
    ```

2. Buat direktori `build`, lalu pindah ke direktori tersebut

    ```bash
    mkdir build
    cd build
    ```

3. Jalankan perintah `cmake` di dalam direktori build

    ```bash
    cmake ..
    ```

4. Build dengan menjalankan `make`

    ```bash
    make
    ```

5. Jalankan executable

    ```bash
    ./main
    ```
